# Project Name


This project implements a simple e-commerce flow with three main modules:
1. **Login** – Users can log in with their credentials.
2. **Cart** – Users can add, remove, and view items in their shopping cart.
3. **Checkout** – Users can proceed to checkout and confirm their order.

It is designed as a basic template for authentication and shopping functionality, suitable for learning or as a foundation for larger applications.
